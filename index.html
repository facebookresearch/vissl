Powered by PyTorch

Built on top of PyTorch which allows using all of its components.

SOTA Self-Supervision methods

Reproducible reference implementation of SOTA self-supervision approaches (like SimCLR, MoCo, PIRL, SwAV etc) and their components that can we reused. Also supports supervised trainings.

Benchmark tasks

Variety of benchmarks tasks (linear image classification, full finetuning, semi-supervised, low-shot, nearest neighbor, object detection) available to evaluate models.

Scalable

Easy to train model on 1-gpu, multi-gpu and multi-node. Seamless scaling to large scale data and model sizes with FP16, LARC etc

Get Started

  1. Install VISSL:

    via conda:
    conda create -n vissl python=3.8
conda activate vissl
conda install -c pytorch pytorch=1.7.1 torchvision cudatoolkit=10.2
conda install -c vissl apex vissl

  2. Try training SimCLR model 

    python tools/run_distributed_engines.py config=test/integration_test/quick_simclr
